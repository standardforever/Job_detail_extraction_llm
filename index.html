Formulas

Total URLs = domains × URLs per domain

Input tokens = total URLs × tokens per URL

LLM input cost = (input tokens ÷ 1,000,000) × input price per 1M

LLM output cost = (output tokens ÷ 1,000,000) × output price per 1M

Total monthly cost = LLM input cost + LLM output cost + monthly server cost

Notes

  • The app requires a minimum of 4 GB RAM to run.
  • The monthly server cost is added as a simple fixed cost.
  • If you want only API cost, set monthly server cost to 0.
